pico-midi

Firmware for building custom MIDI controllers with Raspberry Pi Pico. Supports buttons, touch pads, potentiometers, light sensors, and accelerometers. Configure everything from your browser — no code required.

Getting Started

  1. Download the firmware
    Grab the .uf2 file for your board from the latest release:
    • midictrl-rp2040.uf2 — for Raspberry Pi Pico (RP2040)
    • midictrl-rp2350.uf2 — for Raspberry Pi Pico 2 (RP2350)
  2. Flash it
    Hold the BOOTSEL button on your Pico while plugging it into USB. It will appear as a removable drive. Drag the .uf2 file onto it. The board will reboot automatically.
  3. Configure
    Open the configurator in Chrome or Edge and connect to your device over USB to set up your MIDI mappings.
